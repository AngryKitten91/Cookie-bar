# COOKIE ACCEPT FORM

## FEATURES
- Asks user for permission to set a cookie 
- Sets cookie for one-year period (365 days)
- "Want to know more?" href
- If cookie is set does not display info anymore

------------------

## REQUIRES
- Bootstrap 3.0 / 4.0
- jQuery 
- [js.cookie - link](https://github.com/js-cookie/js-cookie)
